Starting Conferences

Once you start the application, there are only two steps you need to take to start a conference call.

All you need to do is select "Start Conference" from the main menu and then add users into the conference. To do this, you can either press on the screen or select the menu button to display the add users option. You will be able to add multiple users in you list of contacts.
